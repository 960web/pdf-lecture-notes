# Changelog

## [1.0.0] - 2026-05-11

### 初始版本

- 完整 4 阶段工作流：环境准备 → 方案制定 → Pilot 验证 → 批量制作
- 并行 OCR：后台 Agent + PaddleOCR-VL，~10 页/批次
- 通用方案文档：一份方案覆盖所有章节
- 用户交互节点：Phase 0/1/2/3/4 强制确认点
- 边界情况处理：
  - 跨页内容截断（1 页 overlap 机制）
  - PDF 无目录（文本密度/标题采样自动探测）
  - 混合 PDF（文字页直接读 + 图片页 OCR）
  - 中断恢复
  - PDF 加密
  - 页面旋转校正
  - 多栏/复杂排版警告
  - 文件名非法字符清理
  - 磁盘空间估算
  - API 配额预检
  - 数据隐私提醒
- 输出格式支持：Markdown / MD+PDF（整体/分章节）
- 失败重试机制：单批次重试 3 次
